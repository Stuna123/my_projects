Résumé du projet (à mettre dans ton README ou à expliquer à quelqu’un) :

Budget Tracker App est une application web simple qui permet de gérer ses finances personnelles.
L’utilisateur peut ajouter des revenus et des dépenses, 
et l’application affiche le solde total, ainsi que l’historique détaillé des transactions.
Ce projet m’a permis de pratiquer :

- La manipulation du DOM (ajout et suppression d’éléments dynamiquement),
- L’utilisation des événements JavaScript,
- Les calculs dynamiques (solde, revenus, dépenses),
- L'utilisation du localStorage,
- Et la gestion basique de données en tableaux et objets.

C’est un projet parfait pour comprendre comment les données interagissent avec l’interface utilisateur, 
et c’est une excellente base pour pratiquer LocalStorage.
